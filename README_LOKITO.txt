
LOKITO MEGA PACK – Guia Rápido

1. Painel Web (/web_panel):
- Suba no Railway.
- Edite 'agent_config.json' com sua API da OpenAI.
- Acesse o painel, clique nos botões ou digite comandos.

2. Telegram Bot (lokito_telegram.py):
- Crie seu bot em @BotFather
- Insira seu token em 'lokito_telegram.py' no lugar de INSIRA_SEU_TOKEN_AQUI
- Rode o script: python lokito_telegram.py
- Envie comandos no formato: /divulgar [produto] para [público]

3. Spam Simulado (lokito_spammer.py):
- Executa comentários simulados estilo IA.
- Exemplo: python lokito_spammer.py

4. Tradutor Multilíngue (lokito_translate.py):
- Tradução automática com poder viral em várias línguas.
- Exemplo: python lokito_translate.py

Todos os módulos usam IA real via OpenAI. Personalize os textos conforme quiser.
